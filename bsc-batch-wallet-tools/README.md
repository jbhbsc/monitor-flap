# BSC 批量工具

一个纯前端的 BSC 批量操作页面，包含：

- 批量生成钱包，并下载 CSV
- 批量发送 BEP20 代币
- 批量归集 BEP20 代币或 BNB

## 使用

直接用浏览器打开 `index.html` 即可。

链上功能使用 BSC Mainnet RPC：`https://bsc-dataseed.binance.org/`。

## 安全提醒

- 生成钱包建议断网操作，并妥善保存 CSV。
- 私钥只在浏览器本地使用，项目没有后端。
- 批量归集代币时，每个来源钱包都需要有 BNB 支付 gas。
- 批量发送和归集是真实链上交易，请先用小金额测试。
